Оптимізація веб-шрифтів та типографіки

Аспекти роботи з веб-шрифтами та створення якісної типографіки.

Підключення веб-шрифтів

Я використовую кілька підходів до підключення шрифтів:

  • Google Fonts (Roboto та Montserrat)
  • Локальне підключення (Merriweather)
  • Системні шрифти як fallback

Техніки оптимізації

Для покращення продуктивності я застосував:

  1. preconnect для Google Fonts
  2. Тільки необхідні ваги шрифтів
  3. font-display: swap для локальних шрифтів
  4. WOFF2 формат як пріоритетний

Типографічні елементи

Приклад виділеного тексту та тексту з нахилом.

"Читання для розуму - те саме, що фізичні вправи для тіла." Джозеф Аддісон

Приклад коду:

@font-face {
    font-family: 'MyFont';
    src: url('font.woff2') format('woff2'),
         url('font.woff') format('woff');
    font-display: swap;
}
